### GenderCredentialV1

`GenderCredentialV1` is a type extension of credential subject into VC.

### GenderPerson
Gender of a person.

| Property | Type | Required | Notes |
| ---- | ---- | ---- | ---- |
| "@type" | `GenderPerson` | true ||
| gender | `string` | ture ||
